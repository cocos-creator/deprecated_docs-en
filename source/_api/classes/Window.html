Window class for operating editor window

Name Type Description
name String The window name
options Object The options use [BrowserWindow's options](https://github.com/atom/electron/blob/master/docs/api/browser-window.md#new-browserwindowoptions) with the following additional field:
  • 'window-type' String
    Can be one of the list: - dockable: Indicate the window contains a dockable panel - float: Indicate the window is standalone, and float on top. - fixed-size: Indicate the window is standalone, float on top and non-resizable. - quick: Indicate the window will never destroyed, it only hides itself when it close which make it quick to show the next time.
